CSS Memos

  • 重み = !importantが宣言されたCSSルールが優先される
  • 詳細度 = 「詳細」なCSSルールが優先される
  • 宣言順 = 後に宣言されたものが優先される

詳細は「CSS Cascading and Inheritance」を参照のこと。 2016-07-06時点では CSS Cascading and Inheritance Level 4

詳細度に基づいたCSS選択ルール

重みを指定していない場合、CSSの優先度は 「 ID > クラス, 属性([id="hoge"]), 擬似クラス (:hover) > 型 (div) 」 で決まる。

  • CSSルール div .class_name_1 が適用されていれば背景色は赤っぽくなる
  • CSSルール div #class_name_1 が適用されていれば背景色は紫っぽくなる
  • ここではID側が優先され、紫っぽくなる
  • CSSルール div .class_name_1 が適用されていれば背景色は赤っぽくなる
  • CSSルール div .class_name_2 が適用されていれば背景色は緑っぽくなる
  • CSSルール div div div が適用されていれば背景色は青っぽくなる
  • ここではdiv .class_name_2が適用され、緑っぽくなる
